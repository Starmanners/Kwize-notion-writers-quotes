# Notion Kwize Quotes
A minimalist GitHub Pages site that hosts a transparent, dark-mode-responsive Kwize Quote of the Day embed for use inside Notion.

## Features
- Uses your exact Kwize embed settings
- Transparent background
- Auto-switches text color for light/dark mode
- Perfect for embedding into Notion
- Zero dependencies

## How to Use
1. Enable GitHub Pages in your repo settings.
2. Copy the URL GitHub gives you.
3. Paste it into a Notion Embed block.

Done.
